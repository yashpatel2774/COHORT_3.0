Browser Rendering Pipeline

  1. HTML Parsing
  2. DOM Tree Creation
  3. CSS Parsing
  4. CSSOM Tree Creation
  5. Render Tree
  6. Layout
  7. Paint
  8. Composite

Browser first parses HTML and creates the DOM Tree. CSS is parsed into CSSOM Tree. DOM and CSSOM combine to form the Render Tree. Then Layout, Paint and Composite processes are performed to display the webpage.
